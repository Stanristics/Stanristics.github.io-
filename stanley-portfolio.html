Project 03 · 2025 · Full-Stack Product

ONYE ISI — African Hair Marketplace

Solo Build · Germany Launch · AI-Powered Platform

A full-stack marketplace connecting African hair professionals with clients across Germany — built entirely from scratch. Features an AI-powered hair try-on system that composites selected hairstyles onto user-uploaded photos using Cloudflare Workers AI (img2img diffusion), an LLM-powered chat assistant for style advice and bookings, stylist profile creation with photo uploads to Supabase Storage, and a real-time booking system backed by a PostgreSQL database.

Cloudflare Workers AI Stable Diffusion img2img Groq LLM API Supabase PostgreSQL JavaScript HTML / CSS REST APIs Netlify
  • AI hair try-on: user photo + selected style → img2img generation via Cloudflare Workers AI (free tier)
  • LLM chat assistant powered by Groq (llama-3.3-70b) for hair advice and stylist recommendations
  • Supabase backend — stylist profiles, bookings, and sample photos stored in PostgreSQL + Storage buckets
  • Row-level security policies, public storage CDN, and CORS-safe Worker proxy architecture
  • Fully deployed on Netlify — zero infrastructure cost at launch stage
  • Built for the African diaspora in Germany: Berlin, Munich, Hamburg, Frankfurt, Cologne

Project 01 · 2024 · LMU Munich

Automated Data Collection of Rebel Weapon Endowments

w/ Harihara Viaish Ramaiah Sakthivel · Political Science Dept., LMU Munich

A consulting system that leverages LLMs to automate extraction and structuring of complex weapons data from vast unstructured sources — conflict reports, news articles, and field dispatches. Handles mixed data types while performing contextual reasoning to distinguish implicit details like "captured" vs. "purchased" weapons across multilingual corpora at terabyte scale.

LLMs Python NLP Pipelines Data Engineering Political Science Conflict Research
  • Mixed data extraction: numeric (weapon counts), ordinal & string (supplier names)
  • Contextual reasoning — "captured" vs. "purchased" weapon semantics
  • Multilingual document processing at terabyte scale
  • Presented at Ludwig Maximilian University Munich
Project 02 · 2024 · Computational Linguistics

AHAMEFULA — LLMs & Low-Context Languages

Seminar Paper · NLP & African Language Research

A seminar paper analyzing the capability of large language models to understand the underlying structural patterns of low-context languages, using Igbo as a primary case study. Explores how BERT and similar architectures can bridge representation gaps for linguistically rich but underrepresented languages — with implications for cultural preservation and multilingual AI equity.

BERT Python NLP Linguistic Analysis Igbo Language Language Preservation
  • Analyzed LLM capability gaps in low-context, underrepresented languages
  • Case study: Igbo — a rich, explicitly structured African language
  • Applications in cultural heritage and multilingual AI fairness
  • Presented to academic audience; co-authored with research team
